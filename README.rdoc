= Mapel

Mapel is a dead-simple, chainable image-rendering DSL for ImageMagick. 
Still very much an experiment-in-progress, it supports a dozen or so essential
commands.


== Example

  Mapel.render('image.jpg').resize("50%").to('output.jpg').run
